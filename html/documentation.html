BOT

Extends player class, your bot will extend bot class
  • -makeMove(gameMap) -> The function you decide what move your bot makes. Receives a reference to the game map as a parameter
  • -onGameStart(gameMap) -> The function called when the map is first loaded in, for any initial strategy decisions. Receives reference to game map.
  • -mapUpdate() -> The function called after each turn is made. Parameters TBD.
You may add any additional functions / variables, just ensuring that at least makeMove() is implemented.
